# MentorViz - 导师-院系关系可视化工具

## 项目简介
MentorViz 是一款专为保研学生设计的导师-院系关系可视化管理工具。在保研过程中，面对众多院校和心仪导师，如何高效管理信息、分析导师所属院系关系是一大挑战。本工具通过直观的可视化界面，帮助用户：
- 清晰展示导师与院系的从属关系
- 快速筛选和管理心仪导师
- 记录导师详细信息（评分、收藏、联系方式等）
- 批量处理教师信息，提高管理效率

## 功能特性
- **院系管理**：添加、删除、修改院系信息，支持自定义院系名称和简介
- **教师管理**：添加、编辑、删除教师，支持设置评分、收藏、职称、研究方向等
- **批量操作**：通过文件上传或手动输入批量添加/删除教师
- **可视化展示**：可拖拽、调整大小的院系框，自动计算并显示院系交集区域
- **数据持久化**：自动保存数据到本地存储，支持JSON文件导出/导入
- **教师评价**：为导师设置0-100分的评分，标记收藏状态
- **多维度筛选**：支持按院系、职称、评分、收藏状态筛选导师
- **记事本功能**：支持创建多个记事本，记录重要信息
- **教师详情**：支持添加教师主页网址和邮箱，方便快速访问

## 快速开始
1. **克隆仓库**：
   ```bash
   git clone https://github.com/Double-lyb/MentorViz.git
   ```
2. **打开项目**：直接在浏览器中打开 `index.html` 文件
3. **开始使用**：
   - 添加院系：在顶部输入框填写院系名称和简介，点击"添加院系"
   - 添加教师：点击"添加教师"按钮，填写教师信息并选择所属院系
   - 调整布局：拖动院系框或调整大小，查看交集区域
   - 导出数据：点击"导出数据"按钮，保存数据为JSON文件

## 核心功能

### 1. 院系管理
- **添加院系**：输入院系名称和简介，点击"添加院系"按钮
- **删除院系**：从下拉菜单选择要删除的院系，点击"删除院系"按钮
- **修改院系**：点击院系名称或左上角编辑图标，修改院系信息
- **调整布局**：拖动院系框或调整大小，观察交集区域变化

### 2. 教师管理
- **添加教师**：点击"添加教师"按钮，填写姓名、所属院系、职称、研究方向、评分、收藏状态等
- **编辑教师**：点击教师姓名旁的"+"图标，修改教师信息
- **删除教师**：在编辑教师界面点击"删除"按钮
- **批量操作**：
  - 上传文件：选择院系，上传包含教师姓名的TXT文件（一行一个）
  - 手动输入：在文本框中输入教师姓名（一行一个），可设置统一职称

### 3. 数据管理
- **导出数据**：点击"导出数据"按钮，输入文件名后保存为JSON文件（包含教师、院系和记事本数据）
- **加载数据**：点击"加载数据"按钮，选择之前导出的JSON文件
- **本地存储**：所有操作自动保存到本地存储，刷新页面后数据不丢失

### 4. 教师评价与筛选
- **设置评分**：为教师设置0-100分的评分
- **收藏教师**：点击爱心图标标记收藏状态
- **排行榜**：查看按评分排序的教师列表，支持院系、职称、收藏筛选
- **搜索功能**：按院系、职称、评分阈值、收藏状态搜索教师

### 5. 记事本功能
- **创建记事本**：在记事本列表中点击"创建新记事本"
- **编辑笔记**：打开记事本后输入内容，点击"保存"按钮
- **管理记事本**：支持重命名、删除记事本（至少保留一个）

## 技术栈
- **前端**：HTML5 + CSS3 + JavaScript
- **图标**：FontAwesome 6.4.0
- **数据存储**：localStorage（本地存储） + JSON文件（导出/导入）

## 安装与使用
1. **无需安装**：本项目为纯前端应用，直接打开 `index.html` 即可使用
2. **浏览器支持**：推荐使用 Chrome、Firefox、Edge 等现代浏览器
3. **数据安全**：所有数据存储在本地，不会上传到服务器
